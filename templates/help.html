基础使用说明

http://host:8000/s/虚拟路径?key=你的密钥

查看参数

  • &raw=1 纯文本原始内容

  • &mkdir=1 创建目录
  • &upload_url=https://... 从 URL 拉取
  • &content=文本 上传文本（64KB 上限）
  • &delete=1 删除
  • &rename_to=新名称 重命名
  • &move_to=/目标 移动

HTTP 方法

  • PUT /s/{path}?key=xxx 上传（Body 原始内容）
  • POST /s/{path}?key=xxx 上传（multipart）
  • DELETE /s/{path}?key=xxx 删除

完整使用说明

AI Agent FTP 服务。所有写操作经过 SSRF 防护、路径穿越检测和速率限制。

端点

方法路径说明权限
GET/s/{path}?key=<key>查看文件或目录read/list
GET?mkdir=1创建目录write
GET?upload_url=URL从 URL 拉取write
GET?content=数据上传文本write
GET?delete=1删除delete
GET?rename_to=新名重命名rename
GET?move_to=/目标移动rename
PUT/s/{path}?key=<key>上传（Body 原始内容）write
POST/s/{path}?key=<key>上传（multipart）write
DELETE/s/{path}?key=<key>删除delete

通用参数

  • &json=1 JSON 输出
  • &raw=1 原始内容
  • &download=1 下载

安全限制

  • 上传最大 500MB | content= 最大 64KB
  • upload_url 仅 HTTPS，拦截内网（SSRF 防护）
  • 写操作 60 次/分钟（速率限制）
  • 文件名净化，拒绝路径穿越
  • 密钥仅记录前 4 位

受限 Agent 示例（仅 GET）

mkdir:  GET /s/data?key=abc&mkdir=1
upload:  GET /s/data?key=abc&upload_url=https://example.com/file.zip
content: GET /s/data/note.txt?key=abc&content=Hello+World
delete:  GET /s/data/old.txt?key=abc&delete=1
rename:  GET /s/data/old.txt?key=abc&rename_to=new.txt
json:    GET /s/data?key=abc&json=1

管理

  • /admin 管理界面
  • /admin/stats 统计
  • /health 健康检查
